# Excel Sales Dashboard

## Project Overview

This project demonstrates how Microsoft Excel can be used to transform raw sales data into an interactive dashboard for business decision-making.

## Features

- Data Cleaning
- Profit & Profit Margin Calculation
- Pivot Tables
- Pivot Charts
- Interactive Dashboard
- KPI Cards
- Slicers

## KPIs

- Total Revenue
- Total Profit
- Total Orders
- Total Customers
- Profit Margin

## Dashboard Includes

- Revenue by Branch
- Revenue by Product
- Monthly Revenue Trend
- Top Customers
- Interactive Filters

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
